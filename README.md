### `PT-BR`

# Post dev

Sistema simples de cadastro de usuário onde se é informado nome e idade, essas informações são enviadas do frontend para um servidor através de requisições HTTP do tipo POST que processa e armazena esses dados em um banco de dados. O projeto é hospedado em plataformas gratuitas de hospedagem para o acesso e utilização pública sendo estes Railway para o backend e banco de dados e Vercel para o frontend. Este projeto inicial tem como objetivo fazer o cadastro simples de dados de maneira que tenha acesso público por meio de plataformas livres de hospedagem de aplicações.

## Tecnologias Utilizadas

- React
- Node.js
- Mysql

## Acesso ao Sistema

Para acessar o sistema e utiliza-lo é disponibilizado este [https://post-dev-frontend-qnn9.vercel.app/](link) que se encontra hospedado na plataforma Vercel onde é acessado de maneira pública para ser utilizado. Este link é o frontend (interface de usuário da aplicação) que se comunica com o backend e em seguida com o banco de dados.

## Funcionalidades

- Cadastro de usuário (nome e idade)

Simple register user system where it's informed the user's name and age, these informations are send by the frontend to the server by HTTP type POST requisitions that process and stores these data in a database. The project it's hosted in free host plataforms to free and public acess and use, which are Railway to the system backend and database and Vercel to frontend. This initial project have the objective to register in a simple way especified data to have free acess by free hosted plataforms.

- React
- Node.js
- Mysql

## Acess the system

To acess the system and use it's available this [https://post-dev-frontend-qnn9.vercel.app/](link) that is hosted in Vercel's plataform where there's public acess to be used.
